Pengertian HTML

HTML adalah singkatan dari Hypertext Markup Language. Markup languages adalah bahasa komputer yang digunakan untuk menyusun , memformat , atau mendefinisikan hubungan antara berbagai bagian dokumen teks dengan bantuan simbol atau tag yang disisipkan dalam dokumen. HTML merupkan standard bahasa yang digunakan untuk menampilkan document web. Dalam HTML, kita juga dapat menambahkan Cascading Style Sheets (CSS) ataupun JavaScript untuk menambahkan variasi pada konten yang akan kita tampilkan.


Struktur HTML

<!DOCUMENT html> : merupakan tag yang digunakan untuk menyimpan informasi versi dari HTML yang dibuat

<html> </html> : tag ini merupakan dasar root dari elemen html. Tag <html> merupakan parent tag dari semua element

<head> : tag head digunakan untuk menyimpan informasi meta data, judul sebuah halaman, halaman CSS dan lain lain. Tag ini tidak ditampilkan ke browser

<body> : tag body digunakan menampilkan semua data yang meliputi teks, gambar hingga hyperlink Semua konten yang ada dan ditampilkan pada halaman html akan dimuat di dalam tag ini


Tag dan Atribut

Elemen : yaitu objek atau simpul HTML yang membentuk satu kesatuan struktur halaman web.

Tag : yaitu sintaks <nama tag> dan </ nama tag> yang digunakan untuk membuat sebuah elemen HTML. Ada dua jenis tag, closed dan open tag.

Atribut : adalah informasi tambahan untuk elemen HTML yang kita buat , misal atribut lebar dan tinggi pada tag <img>.


Pengertian CSS

CSS adalah bahasa yang digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML. CSS berfungsi untuk memisahkan konten dari tampilan visual web. Selain memasukkan file CSS pada dokumen web, kita juga menyisipkannya CSS secara inline maupun dengan menggunakan attribute style pada elemen yang kita inginkan. Pada file dan inline CSS, kita dapat mereferensikan id ataupun class dari sebuah elemen HTML agar hanya elemen yang kita inginkan saja yang berubah tanpa mempengaruhi elemen lain.
