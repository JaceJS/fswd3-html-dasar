Text Formating

Berikut merupakan macam-macam text formatting yang bisa digunakan dalam css :

  1. Tag <b>...</b> untuk membuat teks tebal
  2. Tag <i>...</i> untuk membuat teks miring
  3. Tag <u>...</u> untuk membuat teks garis bawah
  4. Tag <strong>...</strong> untuk membuat teks penting
  5. Tag <em>...</em> untuk membuat teks menekankan
  6. Tag <mark>...</mark> untuk membuat teks menandai
  7. Tag <small>...</small> untuk membuat teks kecil
  8. Tag <del>...</del> untuk membuat teks mencoret
  9. Tag <ins>...</ins> untuk membuat teks sisipan
  10. Tag <sub>...</sub> untuk membuat pangkat di bawah
  11. Tag <sup>...</sup> untuk membuat pangkat di atas
